# Lokeshwaran V Portfolio

A static, front-end-only software engineering portfolio built with HTML, CSS, and JavaScript.

## Files

- `index.html` - page content and sections
- `styles.css` - responsive visual system
- `script.js` - theme switcher, filters, reveal effects, and canvas background

## Personalize

Update these placeholders first:

- `your-email@example.com`
- LinkedIn and GitHub links
- Resume link

Open `index.html` in a browser to view it. No backend or install step is required.
